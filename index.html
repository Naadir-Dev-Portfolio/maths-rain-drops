Arithmetic Rain Game is a single-file HTML5 brain-training game inspired by the classic "Raindrops" game from Lumosity. It was created by Naadir in 2024, with the help of GPT-4 (OpenAI's language model).

Built with:

  • Three.js (for rendering animated falling arithmetic drops)
  • Vanilla JavaScript, HTML5, and CSS3 (no frameworks)
